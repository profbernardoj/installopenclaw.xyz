# Content Directory

*Drafts for blogs, tweets, videos, podcasts, etc.*

---

## Structure

```
content/
├── README.md (this file)
├── twitter/    — Threads, tweets, replies
├── blog/       — Long-form posts
├── video/      — Scripts for video content
├── podcast/    — Show prep, talking points
└── email/      — Newsletters, announcements
```

## Naming Convention

`[type]-[topic]-[status].md`

Examples:
- `blog-ai-agents-draft.md`
- `twitter-everclaw-launch-ready.md`
- `video-space-settlement-script.md`

Status keywords: `idea`, `outline`, `draft`, `review`, `ready`, `published`

---

*Move to drafts/ for general-purpose drafts that don't fit content categories.*
